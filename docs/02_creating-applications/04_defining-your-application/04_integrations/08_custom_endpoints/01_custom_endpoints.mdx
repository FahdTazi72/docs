The Genesis LCNC Platform can easily have its functionality extended through the creation of custom endpoints, making it easy to integrate the platform with existing systems. This is in addition to the RESTful endpoints provided by the Genesis LCNC Platform.

Likely uses for these custom endpoints include file upload and download, and integration into external authentication systems.
