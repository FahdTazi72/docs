The Genesis Document Management component gives you the ability to upload and download files to and from a variety of file systems.
The storage strategy can be any one of the following:
* Local Storage
* Amazon S3
* Sharepoint

This component comes with one microservice, `FILE_MANAGER`, which manages the APIs used to display the files existing within the storage system of your choice.

Here, we will explain how to install this component into your application and what configurations you need to set up each one of the aforementioned storage strategies.
