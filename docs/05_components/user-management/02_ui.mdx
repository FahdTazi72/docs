# UI Components

With this quick command you can add the auth UI on your app (changing `myapp` to your app name folder):
```shell
npx -y @genesislcap/genx@latest add myapp -x -s genesislcap/pbc-auth-seed
```

It will create a route entitled *User Management*.

![User Management](/img/pbc-user-management-ui/01_user-management.png)

You can check more details about users, profiles, rights/permissions, etc., in the [authorisation](../../../../server/access-control/authorisation#users-profiles-and-right-codes) section.
